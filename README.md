# secret-boot-framework

#### 介绍
secret-boot脚手架(长期框架版本长期更新)

只实现开发中最基础的springboot、mybatis、redis整合

内置丰富的基础工具类，开箱即用

#### 核心依赖
|  组件   | 版本  |
|  ----  | ----  |
| springboot  | 2.0.8.RELEASE |
| mybatis-plus  | 3.0.5 |
| mysql  | 5.7/8.x |
| redis  |  |
